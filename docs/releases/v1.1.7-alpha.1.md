# Alpha 1.1.7

Russian-productized setup, meaningful locale defaults, and external-mailbox-first outbound operations release for the Alpha 1.1 line.

## Included

- Replaces the old instance-level `region profile` stub with real locale behavior by adding meaningful default `regional format` and `time zone` handling in OOBE and per-user Settings, then wiring those preferences into interface date/time formatting instead of storing decorative values that did not affect the product.
- Continues the Russian-productization work by tightening natural Russian copy across setup, auth, settings, mail, and MFA flows, and by closing remaining untranslated UI leaks on the new setup and account surfaces.
- Improves the first-run language and setup experience with cleaner visual presentation, corrected layout behavior, better multilingual greeting coverage, and setup messaging that adapts more honestly to external-mailbox-only deployments.
- Adds the first full `Outbound` and `Reply Ops` operating model, built around external mailbox accounts such as Gmail and Libero instead of treating them as secondary paths.
- Introduces outbound campaigns, steps, enrollments, events, suppressions, recipient state, and mail-thread bindings, together with external-mailbox-aware sender policies, reply funnels, existing-thread continuation, provider-aware pacing, manual task steps, reply branching, and domain/recipient governance.
- Adds the corresponding backend and API surfaces for campaign creation, audience import, preflight, launch control, reply classification, and reply-ops workflows, plus regression coverage for collector/reply-funnel routing and thread-owner mailbox behavior.
- Simplifies the new outbound and reply-ops interface so it leads with concise operational summaries, clearer sectioning, and progressive disclosure for advanced controls instead of presenting every routing and governance detail at equal visual weight.
- Keeps release installer packaging aligned with upstream Launchpad `v1.3.2`, which is the latest published Launchpad release as of April 26, 2026.
