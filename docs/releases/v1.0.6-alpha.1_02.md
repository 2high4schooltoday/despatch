# Alpha 1.0.6_02

Service lifecycle reliability hotfix for install diagnostics and uninstall compatibility.

## Included

- Reduces false post-install failures when `despatch` is still `activating` but health is already reachable.
- Improves post-check diagnostics with service sub-state details and IPv6-safe local health URL handling.
- Extends uninstaller teardown to remove both current `despatch*` and legacy `mailclient*` systemd units/drop-ins.
- Updates diagnose checks to treat transient `activating` state as transitional instead of immediate `APP_DOWN`.
