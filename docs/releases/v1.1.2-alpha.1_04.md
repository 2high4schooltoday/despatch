# Alpha 1.1.2_04

Verified installer and release provenance update for the Alpha 1.1.2 line.

## Included

- Refactors the Launchpad installer workflow into a dedicated packaging pipeline that builds verified standalone installers with embedded official provenance metadata.
- Adds GitHub attestation-backed proof attachment and `launchpad installer verify` validation for the packaged `x86_64`, `arm64`, and universal installer artifacts.
- Makes the universal Launchpad installer the primary release artifact and installation entrypoint, while keeping architecture-specific installers available for targeted deployments.
- Updates the release workflow so application archives are attested too, and ensures the GitHub release record exists before installer provenance is baked into assets.
- Refreshes the README with a stronger product-style introduction, a real getting-started section, explicit universal installer commands, and manual verification guidance.
- Updates the local Launchpad shim so systems without a local CLI point straight to the verified universal installer download.
