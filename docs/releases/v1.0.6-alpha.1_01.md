# Alpha 1.0.6_01

Installer post-check reliability hardening to prevent false failures after successful installs.

## Included

- Adds retry logic for `GET /api/v1/setup/status` during `post_checks` to absorb startup timing delays.
- Fixes local URL construction for IPv6 listen addresses by emitting bracketed HTTP URLs.
- Keeps strict `post_checks` failure behavior for real service or routing errors.
