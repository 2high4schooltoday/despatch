# Alpha 1.1.5

Mail workflow and desktop three-pane refinement release for the Alpha 1.1 line.

## Included

- Adds Undo Send for account-backed send-now drafts by queueing a short grace window, surfacing an explicit Undo action in compose/status, and reopening the draft cleanly if the send is canceled before delivery.
- Adds one-click unsubscribe detection and execution for both live and indexed mail by parsing `List-Unsubscribe` / `List-Unsubscribe-Post`, showing the action only when a message exposes it, and supporting HTTPS one-click plus `mailto:` fallback flows.
- Adds Sweep actions for indexed mail so operators can archive, trash, move, or keep only the newest messages from a sender, with optional future-rule creation for the same cleanup behavior.
- Adds reusable compose snippets plus real favorites and pinning across folders, current views, senders, domains, conversations, and individual messages, with sidebar surfacing and reader/toolbar actions that make those shortcuts easy to reuse.
- Makes the desktop three-pane mail workspace horizontally resizable with guarded splitters and minimum widths so core mailbox and reader controls stay visible instead of clipping under aggressive drags.
- Adds focused regression coverage for the new favorites/snippets/undo-send/unsubscribe/sweep flow and the guarded pane-resize behavior, plus parser unit coverage for unsubscribe header selection.
