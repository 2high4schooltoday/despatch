# Alpha 1.0.9_07

Password reset installer and upgrade-migration hotfix release for the Alpha 1.0.9 line.

## Included

- Makes fresh installs configure a working same-host reset mail path automatically when Despatch is deployed with the local Postfix relay.
- Fixes reset sender defaults by deriving `no-reply@<domain>` from the parent domain when Despatch is hosted on `mail.<domain>`.
- Automatically migrates existing hosts away from the broken loopback `127.0.0.1:587` reset-mail pattern during upgrades, without touching valid custom SMTP setups.
- Adds regression coverage for installer defaults, updater migration behavior, and reset sender fallback handling.
