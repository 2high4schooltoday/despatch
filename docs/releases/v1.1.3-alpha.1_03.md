# Alpha 1.1.3_03

Installer corrective release for the Alpha 1.1.3 line, focused on shipping the actual Launchpad runtime fix level and removing the last install-time diagnostic hard fail.

## Included

- Updates the standalone Launchpad installer packaging workflow to build against Launchpad `v1.3.1`, which is the upstream release that fixes installer progress reporting and stage/progress-bar behavior.
- Updates the packaged Launchpad runtime launchers to `v1.3.1` as well, so the shipped Despatch installer matches the upstream runtime that already behaves correctly.
- Stops the install operation from failing on `combinedHealthDiagnostics.ok` during the final diagnostic pass; the installer now reports the combined result and points operators to `launchpad apps diagnose despatch` for the detailed follow-up instead of aborting the install.
- Adds an installer `install --dry-run` validation step to the packaging workflow so install-surface regressions get exercised before another release is published.
- Keeps the previous exact-host LetsEncrypt validation and stage-structured Launchpad manifest improvements in place.
