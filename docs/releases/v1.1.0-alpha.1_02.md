# Alpha 1.1.0_02

UI layout and control-system reflow hotfix for the Alpha 1.1 line.

## Included

- Reflows the app shell and topbar into a tighter three-zone layout so brand, workspace title, primary tabs, and utility actions align cleanly with less dead space.
- Compresses the Mail header and command area, keeps account health in an anchored disclosure instead of inline layout flow, and tightens reader/message-list structure so content regains visual priority.
- Converts Contacts, Settings Mail, Rules, and Scripts into clearer browse/detail/edit flows with compact empty states, explicit edit mode, simpler navigation rows, and fewer floating controls.
- Normalizes interactive chrome across buttons, menus, inputs, selects, toggles, and repeatable form rows so the IBM Plex Mono machine-dark UI reads consistently without browser-native artifacts.
