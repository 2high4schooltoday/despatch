# Alpha 1.0.11_01

Updater stale-prepare recovery hotfix release for the Alpha 1.0.11 line.

## Included

- Recovers stuck automatic-update prepare requests so Admin no longer stays pinned on `Preparing update`.
- Clears stale prepare queue entries so a manual web install can be queued again.
- Keeps updater status writes working for non-root callers without depending on a local `despatch` system user.
