# Alpha 1.1.7_01

Language-gate persistence and installer-packaging hotfix release for the Alpha 1.1.7 line.

## Included

- Fixes signed-out OOBE language switching when the page had previously been opened with a forced `?lang=...` URL, so choosing a different language in the language gate now persists correctly across reloads instead of snapping back to the old forced language.
- Keeps the guest-language URL override aligned with the explicit OOBE selection, while still dropping the override when the chosen language matches the detected system language.
- Updates Launchpad installer packaging to upstream Launchpad `v1.3.3`, the latest published Launchpad release as of April 28, 2026, bringing the newer installer progress chips, progress-bar clipping fix, steadier repaint cadence during heavy work, and detached `stdin` handling for external installer commands.
