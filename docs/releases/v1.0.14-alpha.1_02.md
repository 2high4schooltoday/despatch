# Alpha 1.0.14_02

Rust dependency security hotfix for the Alpha 1.0.14 line.

## Included

- Upgrades the Rust workspace lockfile so `update_worker` no longer resolves the vulnerable `quinn-proto 0.11.13` path through `reqwest` and `quinn`.
- Keeps the shipped application version and behavior unchanged while rebuilding the worker against `quinn-proto 0.11.14`.
