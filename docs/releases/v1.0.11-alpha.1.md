# Alpha 1.0.11

Mail workflow reliability and mailbox-action release for the Alpha 1.0 line.

## Included

- Adds an in-reader conversation rail so threaded messages can be scanned and opened without leaving the reader pane.
- Makes compose drafts durable across attachment uploads, failed sends, retries, replies, and forwards.
- Adds core mailbox actions in the mail list: archive, move, mark read/unread, flag toggle, and bulk selection.
