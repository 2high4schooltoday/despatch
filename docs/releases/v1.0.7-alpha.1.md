# Alpha 1.0.7

Compose workflow rework release with richer send controls and stricter sender-policy enforcement.

## Included

- Rebuilds compose UI into a layered toolbar + metadata + writing-canvas layout aligned with Despatch theme constraints.
- Adds rich compose capabilities: `Cc/Bcc`, sender identity selection with secure manual fallback, regular attachments, and inline CID images.
- Introduces `GET /api/v1/compose/identities` and extends `POST /api/v1/messages/send` with rich compose fields while keeping backward compatibility.
- Upgrades SMTP/MIME generation for HTML+plain multipart alternatives, inline related parts, and envelope routing across `To/Cc/Bcc`.
- Expands API/mail/UI regression coverage for sender-mode enforcement and compose runtime behavior.
