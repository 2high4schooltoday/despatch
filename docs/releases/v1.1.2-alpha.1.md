# Alpha 1.1.2

Mail triage, conversation reliability, and workflow ergonomics release for the Alpha 1.1 line.

## Included

- Fixes conversation navigation so thread context survives stale live summaries, detail loads, duplicate mailbox rows, and reply/send continuity.
- Adds conversation-level triage with Snooze, Follow Up reminders, Tags, one optional Category, triage smart views, triage filters, inline chips, and `Settings > Mail > Triage` management.
- Reworks the Notification Centre into a theme-aware, lower-noise inbox that keeps routine status chatter out of persistent notifications while surfacing due follow-ups and important system events.
- Tightens Mail workspace ergonomics with denser command bars, better typography, stronger split-view flow, bounded filters, cleaner reader controls, and HIG-aligned navigation/search behavior without changing the established visual style.
- Extends backend and browser coverage for triage, notifications, and mail reliability paths, including full `go test ./...` coverage and targeted Playwright regressions for the new workflows.
