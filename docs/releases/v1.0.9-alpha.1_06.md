# Alpha 1.0.9_06

Updater web-status hardening hotfix release for the Alpha 1.0.9 line.

## Included

- Stops web/admin updater checks from depending on the root-only `lock/update.lock` path.
- Reports queued updater state from request/status files instead of direct lock-file inspection.
- Adds regression coverage for idle, forced-check, pending-request, and stale-queue cases when the lock directory is unreadable.
