# Alpha 1.1.5_01

Release-contract hotfix for the Alpha 1.1.5 line.

## Included

- Fixes the failed `v1.1.5-alpha.1` release workflow by restoring strict UI contract compliance in the mailbox pane splitter focus treatment.
- Replaces the splitter focus `box-shadow` with an outline-based focus indicator so the guarded three-pane resize UI keeps the same intent without tripping the release gate.
- Keeps the Alpha 1.1.5 mail workflow features intact: Undo Send, one-click unsubscribe, Sweep, snippets, favorites/pinning, and guarded desktop pane resizing all remain unchanged apart from this contract-safe focus styling.
