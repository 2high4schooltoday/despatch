# Alpha 1.1.1

Mail reliability and compose tooling release for the Alpha 1.1 line.

## Included

- Restores visible conversations in the reader and hardens thread handling so singleton threads stay visible, stale summaries recover from message detail, and indexed rows always expose a usable `thread_id`.
- Fixes compose scheduled send so `Schedule` persists across autosave races instead of snapping back to `Send now`.
- Adds HTML source compose mode with a native syntax-highlighted editor, line numbers, live iframe preview, safe rich/HTML switching, and draft restore that preserves the selected body mode.
- Makes updater messaging truthful for scheduled installs, adds a persistent Notification Centre, and renders release versions in the human-friendly `Alpha 1.1.1` style instead of raw tags and commit hashes.
- Cleans up Mail account health by hiding unsupported quota actions and quota copy when the server does not expose quota support.
