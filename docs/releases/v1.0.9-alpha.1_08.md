# Alpha 1.0.9_08

Updater trigger-limit and self-recovery hotfix release for the Alpha 1.0.9 line.

## Included

- Fixes updater watcher failures caused by persistent request-file matches driving `despatch-updater.path` into systemd trigger-limit failure.
- Switches updater request watching to request-directory change detection and consumes queued requests earlier in the privileged workers.
- Clears failed updater unit state automatically during install and updater unit refresh, instead of relying on manual `enable --now` recovery.
- Improves Admin diagnostics so trigger-limited updater failures are reported explicitly with the correct repair command.
