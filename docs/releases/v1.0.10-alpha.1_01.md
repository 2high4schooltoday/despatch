# Alpha 1.0.10_01

Updater install and watcher hardening hotfix release for the Alpha 1.0.10 line.

## Included

- Stops updater status and install checks from mutating the watched request directory, preventing self-triggered path-unit failures on fresh installs.
- Starts and verifies `despatch-updater.path` before `despatch`, with final automatic recovery if the watcher drops into a failed state during setup.
- Makes both updater worker entrypoints ignore noisy path-unit wakeups when no real update request exists, instead of failing on full config loading.
- Improves updater diagnostics so `unit-start-limit-hit` is classified correctly and points to the right systemd recovery path.
