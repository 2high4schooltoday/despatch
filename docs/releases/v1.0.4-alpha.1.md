# Despatch Alpha 1.0.4

This alpha release delivers the next major foundation for Despatch with stronger security, broader API surface, and expanded mail workflows.

## Included

- Added `/api/v2` endpoints for accounts, mailboxes, threads, messages, search, drafts, rules, sessions, quota, and crypto/security operations.
- Implemented end-to-end MFA onboarding/enforcement with TOTP + WebAuthn, legacy-user prompt flow, trusted devices, and admin-mandatory MFA during setup.
- Added migrations `007` through `018` covering multi-account data, indexing/search, rules/Sieve, MFA/session controls, crypto metadata, sync/quota, and trusted-device state.
- Added Rust MailSec service/contracts integration for parser/security-sensitive operations.
- Upgraded frontend auth/OOBE flows to enforce MFA stages and improve non-technical setup UX.

## Notes

- Database migrations are required before runtime.
- `/api/v1` remains supported for existing integrations.
