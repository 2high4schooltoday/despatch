# Alpha 1.0.7_01

Compose reliability and hierarchy hotfix for the Alpha 1.0.7 line.

## Included

- Refines compose into a stricter writing-first hierarchy: minimal window actions, compact envelope rows, dominant body canvas, and on-demand formatting tools.
- Implements tokenized recipient handling for `To/Cc/Bcc` with deduplication and subtle invalid-address signaling.
- Fixes sender fallback so `From` is resolved automatically to the authenticated email when identity data is unavailable (no manual retyping required).
- Hardens v1 manual sender handling server-side so missing `from_manual` safely falls back to the authenticated account while preserving anti-spoof checks.
