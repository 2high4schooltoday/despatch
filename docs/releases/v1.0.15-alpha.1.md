# Alpha 1.0.15

Mail workflow and mailbox-management release for the Alpha 1.0 line.

## Included

- Fixes draft save and send failures caused by blank scheduled-send values, and auto-bootstraps indexed mail accounts in PAM mode.
- Restores indexed HTML message rendering, decodes non-ASCII sender names correctly, and blocks invalid sender identities before SMTP delivery.
- Adds real multi-account indexed mail with unified views plus first-class folder create, rename, and delete controls in the web sidebar.
- Prevents false updater failure banners after successful installs when backup archiving crosses filesystem boundaries.
