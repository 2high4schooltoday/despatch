# Alpha 1.0.8_03

Authentication and mail-rendering patch release for the Alpha 1.0.8 line.

## Included

- Fixes password-reset reliability in PAM and SQL flows: reset deep links now open correctly, public reset fails closed until the external sender is confirmed, and PAM helper failures no longer burn valid reset tokens.
- Fixes passkey login UX and policy drift: passkey primary sign-in now defaults on, login no longer asks for a passkey email, the passkey option hides when unavailable, and OOBE can set the initial passkey sign-in policy.
- Fixes HTML mail rendering so message HTML stays visually faithful instead of inheriting or being recolored by the surrounding app or machine theme.
