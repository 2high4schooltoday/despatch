# Alpha 1.1.2_03

Launchpad-native installer parity release for the Alpha 1.1.2 line.

## Included

- Migrates Despatch install, status, diagnose, and uninstall to the Launchpad native project under `packaging/despatch-installer/launchpad.lp`, removing the old Bash and Python installer helpers from the live lifecycle path.
- Restores native PAM reset helper parity by using Launchpad `1.1.1` identity lookup to derive `PAM_RESET_ALLOWED_UID` and `PAM_RESET_ALLOWED_GID`, and by wiring the PAM helper service and socket back into native install/uninstall flows.
- Restores native uninstall parity for the Despatch system account by using Launchpad `1.1.1` service-account removal instead of keeping the previous native-gap workaround.
- Adds a dedicated Launchpad installer packaging workflow that validates the installer project and publishes standalone installer artifacts from the same native manifest.
- Hardens the release pipeline by validating the Launchpad installer with an explicit proxy-mode dry-run configuration that matches the native manifest contract.
- Refreshes Rust release dependencies in the locked build graph, including `tar`, `rand`, and `rustls-webpki`, so the release audit passes on current advisories.
- Updates the public entrypoints and README so `despatch`, `scripts/auto_install.sh`, `scripts/uninstall.sh`, and `scripts/diagnose_access.sh` are compatibility shims into Launchpad rather than separate installer implementations.

## Validation

- `launchpad check packaging/despatch-installer` with Launchpad `v1.1.1`
- `launchpad run packaging/despatch-installer --operation status --dry-run`
- `launchpad run packaging/despatch-installer --operation diagnose --dry-run`
- `launchpad run packaging/despatch-installer --operation install --dry-run --yes --set baseDomain=example.com --set proxyServerName=example.com`
- `launchpad run packaging/despatch-installer --operation uninstall --dry-run --yes --set removeSystemUser=true`
- `launchpad build-installer packaging/despatch-installer /tmp/despatch-installers-111`
