# Alpha 1.0.9_04

Password reset delivery reliability hotfix release for the Alpha 1.0.9 line.

## Included

- Fixes password reset delivery by sending reset mail through a real SMTP submit path with proper RFC822 headers and optional dedicated SMTP authentication.
- Makes PAM public reset readiness depend on real reset-sender transport health, not only external-sender confirmation.
- Improves reset diagnostics with transport-specific sender reasons, Admin mail health visibility, and clearer recovery-email guidance.
