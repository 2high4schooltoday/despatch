# Alpha 1.0.14

Daily-driver mail release for the Alpha 1.0 line.

## Included

- Makes indexed mail the primary workspace path, with visible sync freshness and compatibility fallback when the index is cold.
- Adds header-aware conversation threading, faster mailbox and thread loading, and cached readiness checks to reduce repeated IMAP/SMTP churn.
- Ships saved views, smart mail workflows, recipient suggestions, scheduled send, and draft history restore in compose.
