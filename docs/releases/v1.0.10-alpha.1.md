# Alpha 1.0.10

Mail workflow and first-run upgrade release for the Alpha 1.0 line.

## Included

- Adds real server-side drafts with autosave, crash-text recovery, a true Drafts mailbox, and reopen/edit/send across refresh, logout, and devices.
- Reworks OOBE into a calmer setup assistant with clearer step pacing, appearance choice, automatic-update choice, improved admin account setup, and stronger review/finish flows.
- Redesigns Admin > System software updates around a dedicated update surface and adds real automatic update scheduling with prepared nightly installs.
- Further hardens mailbox, password-reset, and updater reliability across Sent/reply handling, reset delivery, and self-healing updater runtime behavior.
