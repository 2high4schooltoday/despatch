# Alpha 1.0.4_05

Installer/build hotfix for Rust privileged binaries.

## Included

- Installer now bootstraps/refreshes Rust stable toolchain with `rustup` when distro Rust is missing or too old.
- Adds required build dependencies for Rust/OpenSSL vendored paths in auto-install mode.
- Adds retry path for `cargo build` of `pam_reset_helper` and `update_worker`.
- Uses `cargo build --locked` for deterministic installer builds.
